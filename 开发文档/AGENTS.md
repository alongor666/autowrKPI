# agent.md（开发文档）

## 本目录的权威角色
本目录用于定义：
- 全局设计规范（视觉 / 交互 / 可访问性）
- 各业务板块的产品与交互规范
- 决策、需求与问题复盘的治理依据

## 重要权威文件
- 全局设计规范：./全局设计规范.md
  - 定义 CSS 变量、颜色系统、图表样式、交互与可访问性
  - 对 css/ 与 js/ 中的所有 UI 实现具有约束力

## 冲突处理规则
- 若实现（css/js）与《全局设计规范.md》冲突：
  → 以《全局设计规范.md》为准，修正实现
- 若板块文档与《全局设计规范.md》冲突：
  → 设计层规则优先，需在板块文档中显式说明例外

## Agent 在此目录的工作路径
1) 涉及视觉 / 颜色 / 间距 / 图表样式 → 先查《全局设计规范.md》
2) 涉及具体板块 → 再查 板块文档/*
3) 涉及业务口径 → 查 reference/
