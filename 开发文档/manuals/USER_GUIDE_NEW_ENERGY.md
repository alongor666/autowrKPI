# 新能源维度测试说明

## 📋 CSV文件格式要求

确保您的CSV文件包含以下字段之一：
- `is_new_energy_vehicle` (推荐)
- `是否新能源`
- `能源类型`
- `燃料类型`
- `是否新能源车`
- `energy_type`

## 🔍 数据格式示例

### 布尔值格式（推荐）
```csv
is_new_energy_vehicle
true      # 新能源车
false     # 燃油车
true      # 新能源车
```

### 文本格式（兼容）
```csv
是否新能源
新能源车     # 新能源车
燃油车      # 燃油车
新能源车     # 新能源车
```

## 🧪 测试步骤

1. **上传数据**
   - 选择包含新能源字段的CSV文件
   - 等待数据处理完成

2. **使用下钻功能**
   - 点击"添加下钻条件"
   - 在步骤4会看到"是否新能源"维度
   - 可选择：true（新能源）或 false（燃油）

3. **验证结果**
   - 筛选后应只显示对应类型的车辆数据
   - KPI指标会相应变化
   - 图表标题会显示筛选条件

## 📊 分析场景

### 新能源车专项分析
1. 选择"是否新能源" = true
2. 分析新能源车的：
   - 保费规模和占比
   - 赔付率和费用率表现
   - 各机构的新能源业务分布

### 新能源vs燃油车对比
1. 分别筛选新能源车和燃油车
2. 对比两组数据的：
   - 变动成本率差异
   - 业务增长趋势
   - 风险特征表现

## 🔧 故障排除

如果新能源维度没有显示：
1. 检查CSV文件是否包含支持的字段名
2. 确认字段名拼写正确（区分大小写）
3. 验证字段值不为空

如果筛选结果不正确：
1. 检查字段值格式（true/false 或 文本描述）
2. 确认数据中没有空值或特殊字符
3. 验证CSV编码为UTF-8
